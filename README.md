# 🎵 Loja de Discos — Projeto SciTecJr

Protótipo de e-commerce para venda de discos de vinil, desenvolvido em grupo (equipe de 4 pessoas) como projeto de admissão para a empresa júnior **SciTecJr**.

## 📌 Problema

Construir uma loja virtual funcional para exibição e venda de discos de música, cobrindo desde a descoberta de produtos até o processo de compra.

## 🛠️ O que foi feito

- **Descoberta de produtos**: página dedicada (`descoberta.html` / `descoberta.js`) para navegação pelo catálogo
- **Carrinho de compras**: módulo completo de carrinho (`carrinho.html` / `carrinho.js`)
- **Contato**: página de atendimento (`contato.html`)
- **Catálogo de dados**: produtos consumidos a partir de um arquivo estático (`discos.json`), com capas de álbuns renderizadas dinamicamente

## ✅ Resultado

Base funcional de um e-commerce musical, com frontend integrado a um backend Node.js preparado para lidar com rotas HTTP, arquivos estáticos e sessões de usuário.

## ⚙️ Tecnologias

**Backend**: Node.js, Express.js, `express-session`, `body-parser`, `cookie`
**Frontend**: HTML, JavaScript (`app.js`)
**Dados**: JSON

## 👥 Equipe

Projeto desenvolvido em grupo (4 integrantes) para a SciTecJr.
